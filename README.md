#Usage

1. run 'cake build'
2. Open index.html in Safari/Chrome

or 

1. run 'cake buid'
2. coffee app.coffee
3. visit http://localhost:5000/ in Safari/Chrome

#Run qunit

1. run 'cake build'
2. Open test/index.html

#Run cucumber

1. run 'cake build'
2. coffee app.coffee
3. cucumber.js features/sample.feature
